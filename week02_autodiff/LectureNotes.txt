week02_autodiff (в видео только сравнение фреймфорков для deep learning)
	1) Caffe:
		- декларативная парадигма
		- языки Protobuf (для описания структуры сети) и C++ (для оптимизации и добавления пользовательских сетей)
		- легко задеплоить
		- невозможно писать большие сети, т.к. файл .protobuf очень быстро растет
		- хорошо поддерживает CV
		- о численной стабильности неизвестно
	2) Theano (надстройки - Lasagne или Keras)
	    - декларативная парадигма
		- языки Python и C
		- нелегко задеплоить, т.к. непопулярный фреймворк
		- подходит для небольших сетей
		- хуже поддерживает CV
		- присутствует численная стабильность
	3) Tensorflow (надстройки - Slim или Keras)
	    - декларативная парадигма
		- языки Python и C++
		- легко задеплоить
		- подходит для больших сетей
		- хорошо поддерживает CV
		- пока нет численной стабильности
	4) Pytorch
	    - императивная парадигма
		- языки Python и C++
		- нелегко задеплоить, т.к. пока мало сетей в этом фреймворке
		- подходит для больших сетей
		- хуже поддерживает CV
		- пока нет численной стабильности
